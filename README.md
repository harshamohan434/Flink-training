# Flink-training

## Set-up

#### If you're using an IDE like IntelliJ:

Go to Run > Edit Configurations

Add this to VM Options:

    --add-opens java.base/java.util=ALL-UNNAMED --add-opens java.base/java.lang=ALL-UNNAMED
